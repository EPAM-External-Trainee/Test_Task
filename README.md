# Проект "Компьютер"
## В проекте должен быть реализован "компьютер", который выполняет основные функции.
К примеру,
*включение
*выключение
*запуск ОС
*запуск программы и т.д, 

Не раскрывая клиенту деталей выполнения этой операции.
